# Pomodoro-Timer

This is a Pomodoro Timer that allows a user to specify the amount of time they wish to work and break for then after the given time has elapsed a sound will play and the user will be prompted to swap from work to break or break to work.

In additon, the user has the option to add and display the tasks they wish to complete during their work time. For each task the user can give their task a title and a description. Once they have completed a task they can select the task they have completed and mark it as complete. This will block of that task so the user is able to see which they have completed and which they have yet to do.
